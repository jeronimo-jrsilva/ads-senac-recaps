# ADS 2026.1 - Central de Recaps Acadêmicos

Este repositório centraliza os materiais de apoio e resumos consolidados das disciplinas do curso de **Análise e Desenvolvimento de Sistemas (2026.1)**.

O projeto nasceu da necessidade de organizar e destilar o vasto volume de informações compartilhadas em sala de aula, transformando transcrições e anotações em guias de consulta rápida, glossários técnicos e revisões estruturadas.

## 🎯 Objetivo
Prover à turma um material de estudo complementar de alta fidelidade, facilitando a revisão de conceitos complexos em disciplinas como Lógica de Programação, Arquitetura de Computadores e Análise de Conformidade.

## 🛠️ Metodologia (Visão Geral)
Os recaps são gerados através de um fluxo de curadoria técnica que envolve:
1. **Captura:** Registro integral dos pontos abordados em aula.
2. **Processamento & Refino:** Uso de tecnologias de inteligência artificial para estruturação de dados e extração de termos técnicos.
3. **Validação & Curadoria:** Revisão de conteúdo para garantir alinhamento com as diretrizes dos professores.
4. **Publicação:** Consolidação em um dashboard interativo para fácil navegação.
