🚀 Panorama do Semestre

Semana 02

Consolidação e Consciência (Semana 2)

Transição do código superficial para o domínio técnico dos operadores em Python. A semana aprofundou na 'física' da computação (Kernel e Drivers) e na proteção jurídica via LGPD. Consolidamos a lógica de precisão cirúrgica e a responsabilidade civil do desenvolvedor, amarrando técnica e legislação em um fluxo de auditoria.

Semana 01

A Fundação da Área Meio (Semana 1)

Abertura do semestre focada na quebra da barreira entre codar e compreender a infraestrutura. Exploramos a computação como área meio com Prof. Nator, a redundância crítica de hardware com Prof. Thiago e o rigor da conformidade com Prof.ª Rayane. A base para o ecossistema enterprise foi estabelecida através da resiliência.

📚 Glossário (50 termos)

Abstração

Processo de focar nos elementos essenciais de um problema, ignorando detalhes irrelevantes. Wikipédia.

Algoritmo

Sequência finita de instruções bem definidas e não ambíguas. Wikipédia.

Aninhamento

Técnica de colocar uma função dentro de outra, como float(input()), para otimizar o fluxo de processamento. Wikipédia.

ANPD

Autoridade Nacional de Proteção de Dados, órgão responsável por fiscalizar a LGPD no Brasil e aplicar sanções que podem chegar a R$ 50 milhões. Wikipédia.

Área Meio

Tecnologia aplicada como ferramenta de suporte para outros campos de conhecimento.

Arquiteto de Software

Profissional responsável pelas decisões de alto nível sobre a estrutura do sistema.

Bare Metal

Software instalado diretamente no hardware físico. Wikipédia.

Biometria

Dados físicos ou comportamentais únicos usados para identificação segura, classificados como Dados Sensíveis. Wikipédia.

BIOS

Sistema básico de entrada e saída responsável por inicializar o hardware e preparar o ambiente para o S.O. Wikipédia.

Booleano

Tipo de dado lógico binário que assume apenas os valores Verdadeiro (True) ou Falso (False). Wikipédia.

Cache

Camada de memória ultrarrápida interna ao processador usada para acelerar o acesso a dados frequentes. Wikipédia.

Casting

Ato de converter explicitamente um tipo de dado em outro (ex: converter string para inteiro). Wikipédia.

Compliance

Estado de conformidade com regras, leis, normas éticas e regulamentos internos de uma instituição. Wikipédia.

Computação Ubíqua

O conceito de tecnologia integrada ao cotidiano, tornando-se onipresente e invisível ao usuário. Wikipédia.

Data Center

Infraestrutura física para hospedagem de sistemas e dados massivos. Wikipédia.

Debug

O ato de encontrar e corrigir erros (bugs) no código fonte de um programa. Wikipédia.

Direito ao Esquecimento

Direito de impedir a perpetuação de informações irrelevantes ou prejudiciais do passado na internet. Wikipédia.

Docker

Plataforma para criação e implantação de aplicações em containers. Wikipédia.

Driver

Software especializado que atua como tradutor entre o hardware específico e o sistema operacional genérico. Wikipédia.

Equivalência Funcional

Princípio jurídico de paridade entre o meio digital e o físico. Wikipédia.

Extensão Universitária

Atividades acadêmicas integradas à comunidade externa. Wikipédia.

F-Strings

Método moderno de formatação de strings no Python que oferece maior legibilidade e performance. Wikipédia.

Firmware

Software de baixo nível gravado diretamente no hardware para controlar suas funções básicas. Wikipédia.

Fullstack

Desenvolvedor capaz de atuar em todas as camadas da aplicação.

GDB Online

Ambiente de desenvolvimento web sugerido pelo professor para testes rápidos de sintaxe. Wikipédia.

GDPR

Regulamentação europeia de proteção de dados, considerada a mais rigorosa do mundo. Wikipédia.

Hipervisor

Plataforma que permite a execução de múltiplas máquinas virtuais. Wikipédia.

Hot-swap

Característica que permite trocar componentes sem desligar o sistema. Wikipédia.

Input

Comando para captura de dados fornecidos pelo usuário.

Kernel

O núcleo do sistema operacional que gerencia recursos de baixo nível como memória e processamento. Wikipédia.

Kubernetes

Sistema de orquestração para gestão automatizada de containers. Wikipédia.

LaTeX

Motor de formatação de documentos de alta qualidade utilizado na produção dos materiais didáticos do professor. Wikipédia.

Lei de Moore

Observação de que a capacidade de processamento dos computadores dobra aproximadamente a cada dois anos. Wikipédia.

Linguagem de Programação

Método padronizado para comunicar instruções para um computador. Wikipédia.

PEMDAS

Acrônimo para a ordem de precedência (Parênteses, Expoentes, Multiplicação, Divisão, Adição, Subtração). Wikipédia.

Pensamento Algorítmico

Processo de formular problemas cujas soluções podem ser executadas por agentes de processamento.

Portabilidade

Capacidade de mover softwares ou máquinas virtuais entre ambientes distintos.

POST

Power-On Self-Test, o ritual de autodiagnóstico executado pela BIOS no momento do boot. Wikipédia.

RAID

Tecnologia de virtualização de armazenamento para redundância. Wikipédia.

RAM

Random Access Memory, a memória de trabalho volátil onde o processador armazena dados em execução. Wikipédia.

ROM

Read-Only Memory, memória não volátil usada para armazenar instruções fixas, como a BIOS. Wikipédia.

Sintaxe

O conjunto de regras que define a estrutura de escrita correta em uma linguagem de programação. Wikipédia.

SLA

Contrato de garantias mínimas de serviço e disponibilidade. Wikipédia.

Soft Skills

Habilidades interpessoais vitais para o trabalho em equipe. Wikipédia.

SonarQube

Plataforma para inspeção contínua da qualidade e segurança do código. Wikipédia.

Tipagem Dinâmica

O tipo da variável é definido em tempo de execução. Wikipédia.

ULA

Unidade Lógica e Aritmética, componente da CPU onde os cálculos reais são executados. Wikipédia.

Variável

Espaço de memória reservado para armazenar dados mutáveis. Wikipédia.

Violência Patrimonial

Abuso ou controle financeiro indevido exercido sobre os bens ou renda de terceiros. Wikipédia.

VPN

Rede privada segura construída sobre infraestrutura pública. Wikipédia.
